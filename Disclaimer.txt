Introduction/Context:

This Online Retail II data set contains all the transactions for a UK-based online retail between 01/12/2009 and 09/12/2011.The company deals mostly on unique all-occasion gift-ware and its customers mostly wholesalers.

the dataset was downloaded from https://www.kaggle.com/datasets/mashlyn/online-retail-ii-uci

Dataset attributes: 'online_retail' dataset consist of 1,048,576 rows and 8 columns of data.
columns:
InvoiceNo: Invoice number. Nominal. A 6-digit integral number uniquely assigned to each transaction. If this code starts with the letter 'c', it indicates a cancellation.
StockCode: Product (item) code. Nominal. A 5-digit integral number uniquely assigned to each distinct product.
Description: Product (item) name. Nominal.
Quantity: The quantities of each product (item) per transaction. Numeric.
InvoiceDate: Invice date and time. Numeric. The day and time when a transaction was generated.
UnitPrice: Unit price. Numeric. Product price per unit in sterling (Â£).
CustomerID: Customer number. Nominal. A 5-digit integral number uniquely assigned to each customer.
Country: Country name. Nominal. The name of the country where a customer resides.
